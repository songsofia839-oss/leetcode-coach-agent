# LeetCode Coach Agent

An AI-powered study coach built with:

- Google Gemini
- MongoDB Atlas
- Streamlit
- Google Cloud

## Features

- Get coaching for LeetCode problems
- Save study history
- Analyze learning progress
- Recommend next problems to solve

## How it works

1. Enter the name of a LeetCode problem and click "Get Coaching"
